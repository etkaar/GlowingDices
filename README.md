## 2.0 Requirements

- ATtiny85 microcontroller

## 2.0 Layout/Soldering

### Left

  | Color | Use Case |
  | :-- | :-- | :-- |
  | White | Switch position A |
  | Red | Red LED |
  | Black | Ground (3 x LED + 1 x ATTiny85) |
  | Yellow | Ground (Switch) |
  
### Right
  
  | Color | Use Case |
  | :-- | :-- |
  | Red | +5 VDC |
  | White | Switch position B |
  | Green | Green LED |
  | Blue | Blue LED |
